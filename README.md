# Personal Mood Journal App

A Personal Mood Journal App Built with TypeScript

## Features

- Add a new mood entry
- View all mood entries
- View a mood entry by id
- Update a mood entry
- Delete a mood entry

## Technologies Used

- TypeScript
- Node.js
- Eslint
- Prettier

## Installation

1. Clone the repository
2. Run `npm install` to install the dependencies
3. Run `npm run dev` to start the development server

## Usage

1. Open the app in your browser
2. Add a new mood entry
3. View all mood entries
4. View a mood entry by id
